# 🚀 AI Adoption Level and Workforce Dynamics Analysis

This project analyzes the impact of Artificial Intelligence adoption on workforce dynamics using Python and Data Analytics techniques. The analysis focuses on salary changes, job transformation, automation risk, and upskilling trends across different AI adoption levels.

## 📂 Dataset

`/Dataset/ai_job_impact.csv`

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📊 Key Analysis

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Salary Change Analysis
* Automation Risk Analysis
* Workforce Transformation Insights
* Data Visualization

## 📁 Files Included

* `Final_AI_Proj.ipynb`
* `ai_job_impact.csv`

## 📌 Objective

To understand how AI adoption is influencing employees, job roles, salaries, and workforce adaptability across industries.
